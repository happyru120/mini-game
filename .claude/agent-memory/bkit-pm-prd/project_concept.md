프로젝트 컨셉은 "열받은 사람들을 위한 스트레스 해소 파괴형 미니게임 모음"으로 2026-03-26에 완전히 재정의되었다.

**핵심 방향**: 파괴적·카타르시스 게임플레이, 1-5분 짧은 세션, 유쾌한 톤

**게임 목록 (6종, 개발 우선순위 순)**
1. 스트레스볼 연타 (Stress Ball Smash) - 빠른 클릭 연타
2. 접시 날려라 (Dish Smash) - 날아오는 접시 클릭으로 파괴
3. 버블랩 터뜨리기 (Bubble Wrap Pop) - 힐링 모드
4. 분노의 타이핑 (Rage Type) - 타자 속도 + 공감 텍스트 폭발
5. 종이 찢기 (Paper Rip) - 드래그로 서류 찢기
6. 상사한테 던지기 (Boss Throw) - 앵그리버드식 투척

**기술 스택**: 순수 HTML5/CSS3/JavaScript + Canvas API, 백엔드 없음, 60fps 타겟

**핵심 가치**: "Zero Friction Stress Relief" - 설치/로그인 없이 즉시 파괴

**PRD 위치**: docs/00-pm/mini-game-collection.prd.md

**Why**: 기존 PRD는 일반 게임 컬렉션 컨셉이었으나, 스트레스 해소라는 명확한 감정적 니즈에 집중하는 방향으로 전면 재설계됨.

**How to apply**: 이후 기능 추가나 게임 설계 시 "카타르시스 효과가 있는가", "즉각 파괴 피드백이 있는가"를 최우선 기준으로 판단.
